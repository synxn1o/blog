## I. 签证材料清单


参考: [Checklist-for-Study-–-Post-graduation.pdf](https://ambpechino.esteri.it/wp-content/uploads/2026/06/Checklist-for-Study-%E2%80%93-Post-graduation.pdf)


| #   | 材料                           | 状态     | 备注                                                                                                                         |
| --- | ---------------------------- | ------ | -------------------------------------------------------------------------------------------------------------------------- |
| 1   | 国家签证申请表                      | ⬜ 待办   | 英文/意文填写，中文签名                                                                                                               |
| 2   | 近期照片                         | ⬜ 待办   | ICAO 格式 4.0×3.5cm，白底，6个月内                                                                                                  |
| 3   | 有效护照 (原件+2份复印件)              | 🔍 确认中 | 需确认：有效期覆盖离开申根后3个月、2页空白页                                                                                                    |
| 4   | 居住证明                         | 🔍 确认中 | 户口本全页复印件 + 身份证，查询上学期间大学对应领区 ([领馆网络](https://ambpechino.esteri.it/zh/chi-siamo/la-rete-consolare/))，非大学生还需要居住证（需在签证申请前1年签发） |
| 5   | 机票预订单                        | ⬜ 待办   | 可用可退票的预订                                                                                                                   |
| 6   | 学位证书                         | ⬜ 待办   | 原件或公证书 + 海牙认证（Apostille）；如学校要求还需 Declaration of Value 或 CIMEA 认证                                                           |
| 7   | Universitaly 注册证明（Riepilogo） | ⬜ 待办   | 按照学校邮件的指引, 完成Universitaly预注册; 通常需要学位证书及成绩单公证+海牙认证. 注册后需从 universitaly.it 下载 Riepilogo                                      |
| 8   | 派出大学证明信                      | ⏭️ 不适用 | 仅交换项目需要，非交换可跳过                                                                                                             |
| 9   | 资金证明                         | ⬜ 待办   | 需提供：出生公证（海牙认证）+ 父母工作收入证明 + (营业执照副本, 如果有) + 其他收入/奖学金证明; 如果是存款则需要另写一份说明                                                      |
| 10  | 意大利住宿证明                      | ⬜ 待办   | 三选一：学校住宿证明 / 房东邀请信（官网表格）+ 租赁合同 / 酒店确认单（至少1个月）                                                                              |
| 11  | 医疗保险                         | ⬜ 待办   | 需覆盖医疗+住院，符合 T.U. n.286/1998 art.39 要求                                                                                      |
| 12  | 回邮信封                         | ⬜ 待办   | 预填地址 + 贴邮票，用于寄回护照                                                                                                          |
| 13  | 附加材料（可选）                     | ⏭️ 可选  | 任何有助于说明出行目的的补充文件                                                                                                           |

### Checklist

**可复制版本见最后 APPDX A)**


#### 🔴 毕业后才能办（依赖学位证）
- [ ] **学位证书&成绩单公证 + 海牙认证（Apostille）**
  - 需要先拿到毕业证 → 公证处公证 → 外交部海牙认证
  - 成绩单海牙认证可能需要: 学信网成绩单报告
  - 如学校要求：办理 Declaration of Value 或 CIMEA 认证
- [ ] **签证申请表**（临近递交时填写）


#### 🟡 可以提前准备
- [ ] **出生公证 + 海牙认证**（用于资金证明）
- [ ] **护照检查**
  - 确认有效期：离开申根区后 ≥ 3个月
  - 确认 ≥ 2 页空白签证页
  - 准备 2 份复印件（资料页 + 签字页 + 历史申根签证页）
- [ ] **居住证明**
  - 确认大学所在的领区
  - 复印户口本全页
  - 如用居住证：需在申请前 ≥ 1 年签发
- [ ] **资金证明材料**
  - 父母工作收入证明（签字盖章，注明月薪）+ 银行流水 + (营业执照副本, 如有)
  - 或: 存款证明 + 自由职业说明
  - 奖学金证明（如有）
- [ ] **照片**（6个月内拍摄，白底，4.0×3.5cm）


#### 🟢 拿到Pre-enrollment后办
- [ ] **Universitaly 平台预注册** → 获取 Riepilogo 回执
- [ ] **机票预订单** (无需真实下单, 或可退票预定)
- [ ] **住宿证明** (酒店预订单)
  - 优先联系大学获取学校住宿证明
  - 或找房东签邀请信（用官网表格）
- [ ] **医疗保险**（覆盖意大利境内医疗+住院）


#### ⚪ 临近递交需准备
- [ ] **回邮信封**（贴邮票，写好地址）
- [ ] 整理全部材料，检查完整性
- [ ] 预约签证递交时间


### 注意事项

- 签字必须是**中文汉字**，不接受拼音
- 大使馆保留要求补充材料的权利
- 资金证明需提供**完整的经济担保链**（出生公证 → 父母收入 → 其他来源）
- 建议材料齐全后再预约，避免补材料耽误时间
